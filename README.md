# Документация OpenCode (на русском)

Это локальная копия документации OpenCode, переведенная на русский язык.

## Веб-просмотр

Для удобного просмотра документации в браузере:

```cmd
npx serve docs
```

Затем откройте http://localhost:3000

## Содержание

Смотрите навигацию в [docs/_sidebar.md](docs/_sidebar.md) или запустите веб-версию выше.
