# Темы (Themes)

Выберите встроенную тему или определите свою собственную.

## Требования к терминалу (Terminal requirements)

Для корректного отображения тем ваш терминал должен поддерживать **truecolor** (24-bit color).
Проверьте поддержку: `echo $COLORTERM` (должно вывести `truecolor` или `24bit`).

## Встроенные темы (Built-in themes)

OpenCode поставляется с несколькими встроенными темами:
*   `system` (адаптируется к вашей цветовой схеме терминала)
*   `tokyonight`
*   `everforest`
*   `ayu`
*   `catppuccin`
*   `catppuccin-macchiato`
*   `gruvbox`
*   `kanagawa`
*   `nord`
*   `matrix`
*   `one-dark`

## Системная тема (System theme)

Системная тема (`system`) генерирует оттенки серого на основе цвета фона вашего терминала и использует стандартные цвета ANSI (0-15) для подсветки синтаксиса.
Использует `"none"` для текста и фона, чтобы сохранить нативный вид вашего терминала.

## Использование темы (Using a theme)

Вы можете выбрать тему через команду `/theme` или указать ее в конфиге:

```json
{
  "theme": "tokyonight"
}
```

## Пользовательские темы (Custom themes)

### Иерархия
Темы загружаются в следующем порядке (последние переопределяют первые):
1.  Встроенные темы.
2.  Каталог конфига пользователя (`~/.config/opencode/themes/*.json`).
3.  Корневой каталог проекта (`<project-root>/.opencode/themes/*.json`).
4.  Текущий рабочий каталог (`./.opencode/themes/*.json`).

### Создание темы
Создайте JSON файл в одной из директорий тем.

### Формат JSON
Поддерживает:
*   Hex цвета: `"#ffffff"`
*   ANSI цвета: `3` (0-255)
*   Ссылки на цвета: `"primary"`
*   Варианты Dark/Light: `{"dark": "#000", "light": "#fff"}`
*   Нет цвета: `"none"` (наследует цвет терминала)

```json
{
  "$schema": "https://opencode.ai/theme.json",
  "defs": {
    "nord0": "#2E3440"
  },
  "theme": {
    "primary": { "dark": "nord8", "light": "nord10" },
    "text": { "dark": "nord4", "light": "nord0" }
  }
}
```
