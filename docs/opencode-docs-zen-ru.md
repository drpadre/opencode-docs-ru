# Zen

Курируемый список моделей, предоставляемых OpenCode.

## Описание (Background)

Существует огромное количество моделей, но лишь немногие из них хорошо работают в качестве агентов для кодинга. Кроме того, большинство провайдеров настроены по-разному, поэтому вы получаете очень разное качество и производительность.

> **Совет**: Мы протестировали избранную группу моделей и провайдеров, которые хорошо работают с OpenCode.

Поэтому, если вы используете модель через что-то вроде OpenRouter, вы никогда не можете быть уверены, получаете ли вы лучшую версию модели.

Чтобы исправить это, мы сделали несколько вещей:
1.  Мы протестировали избранную группу моделей и поговорили с их командами о том, как лучше всего их запускать.
2.  Затем мы поработали с несколькими провайдерами, чтобы убедиться, что они обслуживаются правильно.
3.  Наконец, мы протестировали комбинацию модель/провайдер и составили список, который мы смело можем рекомендовать.

OpenCode Zen — это AI шлюз, который дает вам доступ к этим моделям.

## Как это работает (How it works)

OpenCode Zen работает как любой другой провайдер в OpenCode.

1.  Вы входите в [OpenCode Zen](https://opencode.ai/auth), добавляете платежные данные и копируете свой API ключ.
2.  Вы запускаете команду `/connect` в TUI, выбираете OpenCode Zen и вставляете свой API ключ.
3.  Запустите `/models` в TUI, чтобы увидеть список рекомендуемых нами моделей.

```bash
/connect
```

Вы платите за запрос, и можете пополнять счет своего аккаунта.

## Эндпоинты (Endpoints)

Вы также можете получить доступ к нашим моделям через следующие API эндпоинты:

*   `https://opencode.ai/zen/v1/responses` (@ai-sdk/openai)
*   `https://opencode.ai/zen/v1/messages` (@ai-sdk/anthropic)
*   `https://opencode.ai/zen/v1/models/gemini-3-pro` (@ai-sdk/google)
*   `https://opencode.ai/zen/v1/chat/completions` (@ai-sdk/openai-compatible)

ID модели в вашей конфигурации OpenCode использует формат `opencode/<model-id>`. Например, для GPT 5.1 Codex вы бы использовали `opencode/gpt-5.1-codex`.

### Модели (Models)

Вы можете получить полный список доступных моделей и их метаданные по адресу:
`https://opencode.ai/zen/v1/models`

## Ценообразование (Pricing)

Мы поддерживаем модель оплаты по мере использования (pay-as-you-go).
> **Примечание**: Комиссии по кредитным картам передаются по себестоимости (4.4% + $0.30 за транзакцию); мы не взимаем ничего сверх этого.

Бесплатные модели (на ограниченное время):
*   Grok Code Fast 1
*   GLM 4.7
*   MiniMax M2.1
*   Big Pickle

### Авто-пополнение (Auto-reload)

Если ваш баланс опускается ниже $5, Zen автоматически пополнит его на $20. Вы можете изменить сумму авто-пополнения или отключить его.

### Ежемесячные лимиты (Monthly limits)

Вы также можете установить ежемесячный лимит использования для всего рабочего пространства и для каждого члена вашей команды.

## Приватность (Privacy)

Все наши модели размещены в США. Наши провайдеры придерживаются политики нулевого хранения (zero-retention) и не используют ваши данные для обучения моделей, за исключением бесплатных моделей в период их тестирования.

## Для команд (For Teams)

Zen также отлично подходит для команд. Вы можете приглашать коллег, назначать роли, курировать модели, которые использует ваша команда, и многое другое.

> **Примечание**: Рабочие пространства в настоящее время бесплатны для команд в рамках бета-тестирования.

### Роли (Roles)

*   **Admin**: Управляет моделями, участниками, API ключами и биллингом.
*   **Member**: Управляет только своими собственными API ключами.

### Доступ к моделям (Model access)

Администраторы могут включать или отключать конкретные модели для рабочего пространства.

### Bring your own key

Вы можете использовать свои собственные ключи API OpenAI или Anthropic, сохраняя доступ к другим моделям в Zen.

## Цели (Goals)

Мы создали OpenCode Zen, чтобы:
1.  Оценить лучшие модели/провайдеры для агентов кодинга.
2.  Иметь доступ к вариантам высочайшего качества.
3.  Передавать любые снижения цен, продавая по себестоимости.
4.  Не иметь привязки (lock-in), позволяя вам использовать его с любым другим агентом.
